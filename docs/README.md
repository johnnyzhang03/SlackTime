# Demo assets

Place `demo.gif` here — a short screen recording of the SlackTime toast
sliding in. The README references `docs/demo.gif`.

How to record one:
- **ScreenToGif** (https://www.screentogif.com/) — easiest for a clean GIF
- **Xbox Game Bar** (Win+G) — records MP4; convert to GIF if desired

Trigger the toast for the recording via the tray icon → **Remind now**.
Keep it under ~5 seconds and a few MB so it loads fast on GitHub.
